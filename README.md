# homebrew-magicadventre

Homebrew tap for **Magic Adventure**, a terminal RPG built in Go.

## Install

```bash
brew tap MaizerGomes/magicadvanture
brew install magicadventure
```

## Usage

```bash
magicadventure
```

## Uninstall

```bash
brew remove magicadventure
brew untap MaizerGomes/magicadvanture
```

For the full game, visit: https://github.com/MaizerGomes/magicadvanture
